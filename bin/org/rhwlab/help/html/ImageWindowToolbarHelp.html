ImageWindow toolbar help

This covers the toolbar on the ImageWindow and the EditImage window. For help on the ImageWindow itself see Help..AceTreeDemo help on the main AceTree control menu.

There are four buttons on the toolbar in its fully filled out form.

  • Help button
  • Properties button
  • AddOne button
  • AddSeries button

Usage notes (under construction)

Here is an example of usage in manual editing.
  • bring up the AddOne dialog by clicking the button.
  • On the dialog, set the radio button to addCell
  • drop cell by left clicking at the proper plane and xy location
  • rebuildAndRename to put that cell on the root using the button
  • close the AddOne dialog using the "x" box
  • bring up AddSeries dialog
  • right clock on the Nuc you just created to make it the starting cell of a series
  • move to the next or subsequent time point using keyboard arrow keys
  • set radio buttons for add cell on the AddSeries dialog
  • left click where the continuing cell should go
  • it will be named the same as its parent
  • continue moving forward in time, dropping cells as desired
  • note that you may change cell size or location but that will only change the cell you added last, not the interpolated cells. Interpolated cells in the next series will be sized like the one you have now.
  • when the series completes:
  • if it is to link to an existing cell, set the radio buttons to Link and right click on the cell to link to
  • In any case, click the rebuildAndRename button
  • Undo is not offered here: use KillCells to undo
