sgt_v30.jar

To use this feature you must obtain sgt_v30.jar.
Please visit: http://www.epic.noaa.gov/java/sgt/

Put it where your copy of AceTree.jar is and try:
java -jar AceTree.jar

If your version is different from sgt-v30.jar you may need to add it to your classpath and use a different command, e.g.,
java -cp AceTree.jar:ij.jar:sgt_v40.jar AceTreeLauncher
